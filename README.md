# Proyecto Final — Molino de Viento 3D

## Requisitos previos

- **Python 3.10+** — [descargar](https://www.python.org/downloads/)
- **Node.js 18+** — [descargar](https://nodejs.org/)

## Ejecutar el Visor Web

```bash
cd web-viewer
npm install
npm run dev
```

Abre en el navegador: **http://localhost:5173**
